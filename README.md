# 🚀 Siber Projesi

> Güvenlik ve teknoloji için tasarlanmış modern web uygulaması

---

## 📋 Proje Hakkında

Bu proje, veri yönetimi ve görselleştirme için geliştirilmiş tam yığın bir web uygulamasıdır. Python backend'i, SQL veritabanı ve dinamik frontend bileşenleriyle güçlü bir altyapı sunar.

---

## 🛠️ Teknoloji Yığını

| Teknoloji | Amaç |

| **Python** | Backend ve API geliştirme |
| **SQL** | Veri tabanı yönetimi |
| **JavaScript** | Frontend etkileşimler |
| **HTML/CSS** | Kullanıcı arayüzü |

## 🚀 Kurulum ve Çalıştırma

### Gereksinimler
- Python 3.8+
- SQLite veya MySQL
- Modern web tarayıcısı

### Adımlar

1. **Depoyu klonla:**
   ```bash
   git clone <repository-url>
   cd siber
   ```

2. **Veritabanını kur:**
   ```bash
   sqlite3 database.db < eylül.sql
   ```

3. **Python bağımlılıklarını yükle:**
   ```bash
   pip install flask
   ```

4. **Uygulamayı çalıştır:**
   ```bash
   python app.py
   ```

5. **Tarayıcıda aç:**
   ```
   http://localhost:5000
   ```

## 📖 Kullanım

### Dashboard
Kontrol paneline `dashboard.html` üzerinden erişin ve tüm metrikleri görüntüleyin.

### Kullanıcı Yönetimi
`user.html` sayfasında kullanıcı işlemlerini yapabilirsiniz.

### API Endpoints
- `GET /` - Ana sayfa
- `GET /dashboard` - Kontrol paneli
- `POST /api/users` - Yeni kullanıcı oluştur
